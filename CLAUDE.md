# ScreenCaptureMaintenance

## Workflow par défaut

Utiliser les skills **superpowers** à chaque tâche, sauf indication contraire explicite de l'utilisateur.

Cela inclut notamment :
- `superpowers:brainstorming` avant toute création ou modification de fonctionnalité
- `superpowers:systematic-debugging` lors de tout bug ou comportement inattendu
- `superpowers:test-driven-development` avant d'écrire du code d'implémentation
- `superpowers:writing-plans` pour les tâches multi-étapes
- `superpowers:verification-before-completion` avant de clore une tâche
- `superpowers:finishing-a-development-branch` en fin de branche
